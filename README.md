# hsaghir.github.io

Source for my personal blog — **[hsaghir.github.io](https://hsaghir.github.io)**.

## Dev

```bash
npm install
npm run dev        # http://localhost:4321
npm run build      # outputs to dist/
npm run preview
```

Requires Node 20+.

Posts live under `src/content/posts/`. Push to `master` deploys via GitHub Actions.
